Miradle


Mim                   Do            Re          Mim
Tened los mismos sentimientos y actitudes del Señor.

( Despojado, siervo humilde:                                    )
( dame un corazón puro, con espíritu firme,                     )
( dispuesto a venderlo todo, pronto para servir (bis).          )

( Obediente hasta la muerte en la cruz:                         )
( dame un corazón noble, con espíritu generoso,                 )
( firme en la contrariedad, en el Padre abandonado (bis).       )

( Amor compasivo:                                               )
( dame un corazón bondadoso, maduro en el amor                  )
( por la fuerza de tu Espíritu, afincado entre los tuyos (bis). )

  Sol          Re
Miradle y quedaréis radiantes,
Do      Sol              Re
será perpetua vuestra alegría. (bis)
